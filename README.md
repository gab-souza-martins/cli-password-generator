This CLI tool can generate either scrambled passwords or passphrases, both configurable by the user.

The word list used for the passphrases was made by [dwyl](https://github.com/dwyl/english-words) and revised by [gabrielweredyk](https://github.com/dwyl/english-words/issues/63#issuecomment-483848840) and [jcameron-au](https://github.com/dwyl/english-words/issues/63#issuecomment-486965298)

# Usage

Simply type `pwd-gen` into your terminal of choice. The tool will ask what kind of password you want, as well as a few options. Type in your answers, press enter after each, and then copy the result.

The tool will store your password settings for your next usages. If you want to change them, type `pwd-gen -c` or `pwd-gen --config` and type in new answers.
